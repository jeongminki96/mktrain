## 모닝브루 포지셔닝 문장 3개

> 경쟁 대안 분석을 바탕으로, 모닝브루가 선택받기 위한 핵심 포지셔닝 제안.

**① 출근 전 5분, 망원동 골목 끝에서 오늘 하루를 시작하는 한 잔**  
→ '빠른 픽업'이 아닌 '짧지만 나를 위한 루틴'으로 포지셔닝. 스타벅스·편의점이 줄 수 없는 공간적 전환감을 선점.

**② 오늘 딱 한 가지 원두, 이유가 있는 커피**  
→ 메가커피·이디야와 다른 이유를 한 문장으로 설명. '오늘의 원두' 스토리를 인스타·칠판으로 매일 공유해 방문 전 기대감 형성.

**③ 바리스타가 기억하는 당신의 취향**  
→ 프랜차이즈가 절대 줄 수 없는 것: 단골 관계. 이름·취향을 기억하는 바리스타 경험을 브랜드 정체성의 중심으로.

## 경쟁 구도 개요

망원동 카페 시장은 크게 세 층으로 나뉜다.

```
[1층] 동네 스페셜티·핸드드립 카페  ← 모닝브루의 직접 경쟁 영역
      프레피커피랩 / 에프비커피로스터즈 / 피피커피 / 루틸 / 빈익스 등

[2층] 합정·망원역 인근 프랜차이즈  ← 출근길 이탈 목적지
      스타벅스 망원역점 / 스타벅스 망원한강공원점

[3층] 가성비 저가 체인 + 편의점   ← 대기 이탈 시 대안
      메가커피 / 컴포즈커피 / GS25·CU 캔커피
```

## 간접 경쟁사 — 저가 체인·편의점

블로그 리뷰에서 고객 대안으로 반복 언급된 곳. 현장 검색에서 망원동 내 지점 직접 확인은 안 됐으나, 인근(합정·홍대 방면)에 다수 위치. `[검색 확인]`

| 경쟁사 | 아메리카노 가격 | 핵심 강점 | 모닝브루와 관계 |
|--------|--------------|---------|--------------|
| 메가커피 | 1,500원 | 초저가, 빠른 대기 | 대기 이탈 고객의 즉시 대안 |
| 컴포즈커피 | 1,500원 | 초저가, 접근성 | 동일 |
| GS25·CU 캔커피 | 1,000~2,000원 | 24시간·결제 편리 | 피크타임 이탈 고객의 최후 대안 |

**핵심 인사이트:** 저가 체인은 "커피 품질"이 아닌 "시간과 가격"으로 경쟁. 모닝브루가 사전 주문 또는 대기 시간 단축을 제공하면 이 이탈을 상당 부분 막을 수 있음. `[customer-pain-points.md 연계]`

## 벤치마킹 제안

### 즉시 참고 가능한 것들

1. **프레피커피랩의 원두 큐레이션 공유 방식** — 핸드드립 메뉴를 "블랙 / 밀크" 두 가지 스타일로 단순화해 주문 장벽 낮춤. 모닝브루 "오늘의 원두" 설명에 같은 방식 적용 가능.

2. **루틸의 스마트스토어** — 오프라인 단골을 온라인 원두 구매 고객으로 전환. 모닝브루가 원두 판매를 한다면 네이버 스마트스토어가 초기 채널로 진입 장벽 낮음.

3. **에프비커피로스터즈의 인스타 ID 활용** — @fbcoffee처럼 기억하기 쉬운 계정명과 일관된 콘텐츠가 검색 유입에 직접 기여.

### 현장 방문 권장 경쟁사 (우선순위 순)

```
1순위: 프레피커피랩 — 망원동 핸드드립 카페 최다 노출, 가격·공간·메뉴 구성 직접 확인
2순위: 피피커피 — 융드립 차별화 전략·원두 소매 모델 확인
3순위: 에프비커피로스터즈 — 커피 매니아 타깃 포지셔닝 관찰
```
