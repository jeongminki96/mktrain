## 1. SEO 진단

### 1-1. `<head>` 메타 영역

| 항목 | 현재 상태 | 평가 |
|------|-----------|------|
| `<title>` | "모닝브루 — 망원동 핸드드립 카페" | ✅ 양호 (브랜드+지역+카테고리) |
| `<meta description>` | **없음** | ❌ 치명적 누락 |
| `<meta robots>` | **없음** | ⚠️ 기본값 허용이지만 명시 권장 |
| `canonical` | **없음** | ❌ 중복 색인 위험 |
| OG (Open Graph) | **없음** | ❌ SNS 공유 시 미리보기 없음 |
| Twitter Card | **없음** | ❌ 누락 |
| `lang="ko"` | 있음 | ✅ 정상 |
| `viewport` | 있음 | ✅ 정상 |

### 1-2. 헤딩 구조 (H1 ~ H3)

| 태그 | 현재 텍스트 | 평가 |
|------|------------|------|
| H1 | "모닝브루" | ❌ 브랜드명만, 핵심 키워드("핸드드립 카페", "망원동") 부재 |
| H2 | 오늘의 메뉴 / 영업시간 / 오시는 길 / 픽업 예약 | ⚠️ 콘텐츠 키워드 약함 |
| H3 | 예약 추가 / 예약 목록 | ⚠️ SEO 가치 낮은 내부 UI 텍스트 |

### 1-3. 콘텐츠 & 키워드

| 항목 | 현재 상태 | 평가 |
|------|-----------|------|
| 핵심 키워드 밀도 | "핸드드립", "망원동", "스콘", "원두" 언급 있음 | ⚠️ 얕음, 설명 텍스트 부족 |
| 이미지 | 없음 (이모지·SVG만 사용) | ⚠️ 이미지 alt 최적화 불가 |
| 내부 링크 | 앵커 링크만 (#menu, #info, #reservation) | ⚠️ 단일 페이지라 구조적 한계 |
| 외부 링크 | Instagram, 카카오채널, 전화 | ✅ rel="noopener" 처리됨 |
| 텍스트 분량 | 전체 가시 텍스트 약 300자 미만 | ❌ 심각하게 얇음 (Thin Content) |

## 2. GEO 진단

> **GEO(Generative Engine Optimization)**: ChatGPT, Perplexity, Google AI Overview 등 생성형 AI가 답변을 생성할 때 이 페이지를 출처로 인용하거나 정보를 추출할 수 있도록 최적화하는 작업.

### 2-1. 엔티티 명확성 (Entity Clarity)

AI가 "모닝브루는 어떤 곳인가?"를 답하려면 아래 항목이 명확해야 한다.

| 항목 | 현재 상태 | 평가 |
|------|-----------|------|
| 상호명 | 모닝브루 (페이지 전체에 일관) | ✅ 명확 |
| 업종 | "핸드드립 카페" (title에만) | ⚠️ 본문에서 반복 필요 |
| 위치 | "서울 망원동 골목 끝" | ❌ 도로명 주소 없음 → AI 인용 불가 |
| 영업시간 | 화~일 08:00–20:00 | ✅ 추출 가능 |
| 전화번호 | 02-123-4567 (가번호) | ❌ 실번호 필요 |
| 메뉴·가격 | 4가지 + 가격 명시 | ✅ 추출 가능 |
| 고유 강점 | "매일 구운 스콘", "핸드드립" | ⚠️ 설명 너무 짧음 |

### 2-2. 구조화 데이터 (AI 신뢰도)

생성형 AI는 JSON-LD 스키마를 직접 파싱해 사실 정보를 추출한다.

| 스키마 타입 | 현재 | 필요성 |
|------------|------|--------|
| `LocalBusiness` / `CafeOrCoffeeShop` | ❌ 없음 | 🔴 최우선 — 위치·시간·연락처 구조화 |
| `Menu` | ❌ 없음 | 🟡 메뉴·가격 AI 추출 가능하게 |
| `FAQPage` | ❌ 없음 | 🟡 AI 답변 직접 인용 가능 |

### 2-3. FAQ / Q&A 콘텐츠

AI Overview, Perplexity는 "망원동 카페 추천", "망원동 핸드드립 어디가 좋아?"에 답할 때 FAQ 형식 텍스트를 인용 우선순위에 둔다.

| 예상 질문 | 현재 답변 가능 여부 |
|----------|-------------------|
| "모닝브루는 몇 시에 열어요?" | ⚠️ 본문에 있으나 구조화 안 됨 |
| "망원역에서 모닝브루 가는 법" | ⚠️ "2번 출구 도보 6분" 있으나 주소 없음 |
| "모닝브루 픽업 예약 방법" | ❌ 설명 텍스트 없음 |
| "모닝브루 오늘의 원두" | ❌ 정적 텍스트로 답변 불가 |
| "망원동 핸드드립 카페 추천" | ❌ 비교 우위 설명 없음 |

### 2-4. 신뢰 신호 (E-E-A-T)

Google의 E-E-A-T(경험·전문성·권위·신뢰)는 AI 인용 여부에도 영향을 준다.

| 항목 | 현재 상태 | 평가 |
|------|-----------|------|
| 운영자 정보 | 없음 | ❌ |
| 리뷰·평점 | 없음 | ❌ |
| 원두 산지·로스팅 정보 | 없음 | ❌ |
| 수상·인증 | 없음 | ❌ |
| SNS 채널 연결 | Instagram 링크 있음 | ✅ |
